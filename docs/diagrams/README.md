# Diagrams Index

This folder contains reusable Mermaid source files for the Healthcare AI demo in three presentation styles:

1. Technical version for engineers and architecture reviews.
2. Investor-friendly version for business and funding conversations.
3. LinkedIn carousel version for short-form visual storytelling.

## Technical Version

| Diagram | File | Best For |
|---|---|---|
| Live Demo Flow | [live-demo-flow.mmd](live-demo-flow.mmd) | Product demos, technical walkthroughs |
| Architecture in One Breath | [architecture-one-breath.mmd](architecture-one-breath.mmd) | Architecture overviews and engineering discussions |
| Reliability / Trust Signal | [reliability-trust-signal.mmd](reliability-trust-signal.mmd) | CI/CD, quality gates, release confidence |

## Investor-Friendly Version

| Diagram | File | Best For |
|---|---|---|

## Rendering Notes

- All files use Mermaid `.mmd` source.
- You can render them in Mermaid-compatible markdown viewers, VS Code Mermaid preview extensions, or export them to PNG/SVG for decks and social posts.
- Keep the `.mmd` files as source of truth and generate image exports from them when needed.
