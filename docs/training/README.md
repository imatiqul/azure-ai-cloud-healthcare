# Training Course Hub

This folder turns the Azure AI Cloud Healthcare codebase into a virtual instructor-led training package suitable for LinkedIn training, enterprise cohort delivery, or internal enablement.

## Training Documents

| Document | Purpose |
|---|---|
| [LinkedIn Training Course Brief](linkedin-training-course-brief.md) | Instructor-facing submission brief with market positioning, audience, proof points, and production requirements |
| [LinkedIn VILT Course Plan](linkedin-vilt-course-plan.md) | Formal 12-week course positioning, outcomes, curriculum map, and 90-day production roadmap |
| [Weeks 1 to 4 Instructor Pack](linkedin-vilt-weeks-1-4-instructor-pack.md) | Detailed slide outlines and guided lab instructions for the first month of delivery |
| [Session-by-Session Lesson Plans](linkedin-vilt-session-plans.md) | Weekly live session, lab, office hour, and homework breakdown anchored to the repo |

## Suggested Use

1. Start with the formal course plan to align the cohort objective, audience, and learning outcomes.
2. Use the LinkedIn course brief when packaging the course for LinkedIn Training, partners, or internal curriculum approval.
3. Use the weeks 1 to 4 instructor pack to accelerate the first month of delivery with ready-made slide and lab structure.
4. Use the session plans to extend the same pattern across the remaining weeks.
5. Reuse the diagrams and stakeholder docs as lecture pre-reads and teaching visuals.
